# OPIc IH 준비 MOC

목표 등급: **IH**

핵심 전략은 많은 주제를 따로 외우는 것이 아니라, 선택 서베이를 서로 엮어서 같은 에피소드를 여러 질문에 재사용하는 것이다. 낮은 배점의 묘사형에서는 힘을 빼고, **기억에 남는 경험, 롤플레이 문제 해결, 비교/이슈 문항**에 발화 에너지를 집중한다.

## 공부 진행 순서

공부 순서는 **스크립트 암기보다 `질문 유형 -> 꺼낼 스토리 -> 시제 -> 문장장치` 순서**로 진행한다.

1. 전체 지도 확인: [[_MOC]]
2. 서베이와 내 캐릭터 고정: [[01-백그라운드-서베이-12개]]
3. 만능 스토리라인 4개 암기: [[04-만능-스토리라인-콤보]]
   - 공원 + 걷기/조깅 + 국내여행/휴가
   - 영화보기 + 집에서 보내는 휴가 + 음악/카페
   - 카페 + 음악 감상 + 요리
   - 해변 + 국내여행 + 해외여행
4. 출제 순서와 시제 감각 잡기: [[02-난이도-5-6-출제-메커니즘]], [[03-10가지-문제유형-시제공략]]
5. 답변 틀 입에 붙이기: [[06-답변-프레임-템플릿]]
6. IH용 문장장치 끼워 넣기: [[05-IH-고득점-문장장치]]

   - 관계사

## 7일 추천 루틴

|  날짜 | 집중할 것                    | 목표                      |
| --: | ------------------------ | ----------------------- |
| 1일차 | 서베이 12개 + 공원/영화/카페/해변 콤보 | 내 캐릭터와 만능 소재 고정         |
| 2일차 | 묘사/루틴 유형                 | 현재 시제로 45~60초 말하기       |
| 3일차 | 최근 경험/기억 경험              | 과거 시제로 1분~1분 30초 말하기    |
| 4일차 | 관계사/현재완료/would           | 각 답변에 문장장치 1개 이상 넣기     |
| 5일차 | 11~13번 롤플레이              | 질문 3개, 대안 2개, 유사 경험 말하기 |
| 6일차 | 14~15번 심화                | 비교/대조와 이슈 답변 구조 만들기     |
| 7일차 | 15문항 모의고사                | 녹음 후 시제, 침묵, 발화량 점검     |

하루 공부 비율:

- 공부 20분
- 말하기 40분
- 녹음 피드백 20분

오픽은 읽는 시간보다 실제로 입으로 말한 시간이 중요하다.

## 시험 운영 원칙

1. 자기소개는 워밍업이다. 길게 힘주지 않는다.
2. 유형 1~2는 현재 시제 중심으로 짧고 안정적으로 답한다.
3. 유형 3~4는 과거 시제와 육하원칙을 초반에 명확히 던진다.
4. 11~13번 롤플레이는 정보 요청보다 **문제 해결과 대안 제시**가 중요하다.
5. 14~15번은 IH와 AL을 가르는 구간이다. 비교 후 개인 의견, 이슈 후 구체 예시로 닫는다.
6. 답변마다 필러를 넣되, 시간을 끌기 위한 침묵 대신 자연스러운 연결어로 사용한다.
