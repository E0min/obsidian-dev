# IH 연습루틴 체크리스트

목표는 원고를 완벽히 외우는 것이 아니라, 질문을 들었을 때 **어떤 콤보를 꺼낼지 3초 안에 결정**하는 상태다.

## 7일 압축 루틴

### Day 1. 서베이와 캐릭터 고정

- [[01-백그라운드-서베이-12개]] 암기
- 내 캐릭터 한 문장으로 말하기
- 공원/영화/카페/해변 콤보를 한국어로 1분씩 말하기

체크:

- [ ] 선택 주제 12개를 보지 않고 말할 수 있다.
- [ ] `운동 안 함`과 `걷기/조깅`을 자연스럽게 설명할 수 있다.

### Day 2. 묘사/루틴 유형

- [[06-답변-프레임-템플릿]] 유형 1, 2 연습
- 공원, 카페, 요리, 음악 각각 45초 답변

체크:

- [ ] 현재 시제로 답변을 유지했다.
- [ ] first, then, after that을 자연스럽게 썼다.
- [ ] 답변 마지막을 감정으로 닫았다.

### Day 3. 최근 경험/기억 경험

- 유형 3, 4 집중
- 해변 여행, 공원 산책, 카페 방문 경험을 각각 1분 30초로 말하기

체크:

- [ ] 답변 초반에 언제, 어디서, 누구와를 말했다.
- [ ] 과거 시제를 유지했다.
- [ ] 예상 밖 사건이나 감정 변화가 들어갔다.

### Day 4. 문장장치 삽입

- [[05-IH-고득점-문장장치]] 연습
- 각 답변에 관계사 1개, 현재완료 1개, would 1개 넣기

체크:

- [ ] where / who / that / which 중 하나를 썼다.
- [ ] have always liked / have been into / has become 중 하나를 썼다.
- [ ] would rather / would usually / I would say 중 하나를 썼다.

### Day 5. 롤플레이

- [[07-롤플레이-11-13번-전략]] 연습
- 예약 문제, 날씨 문제, 물건 고장 문제 3세트 녹음

체크:

- [ ] 11번에서 질문 3개 이상 했다.
- [ ] 12번에서 대안 2개를 제시했다.
- [ ] 13번에서 연기 톤이 아니라 경험 말하기 톤으로 전환했다.

### Day 6. 심화 14-15번

- [[08-심화-14-15번-전략]] 연습
- 스마트폰, 비용 상승, 환경 이슈를 각각 1분 30초로 말하기

체크:

- [ ] 비교 문항에서 A와 B를 분리해 말했다.
- [ ] 이슈 문항에서 구체 예시를 넣었다.
- [ ] 마지막에 개인 의견을 말했다.

### Day 7. 모의고사

- 15문항 흐름으로 녹음
- 답변 길이와 시제만 점검
- 막혔던 질문만 다시 정리

체크:

- [ ] 15문항 전체를 멈추지 않고 끝냈다.
- [ ] 유형 4, 7, 8, 9, 10에서 발화량을 더 많이 가져갔다.
- [ ] 침묵 대신 필러로 시간을 벌었다.

## 매 답변 체크리스트

- [ ] 질문 유형을 먼저 파악했다.
- [ ] 시제를 맞췄다.
- [ ] 첫 문장에서 핵심 답을 했다.
- [ ] 필러를 자연스럽게 사용했다.
- [ ] 관계사/현재완료/would 중 하나 이상을 넣었다.
- [ ] 마지막 문장을 감정이나 의견으로 닫았다.

## 녹음 후 자가 피드백

좋은 답변:

- 질문에 바로 답한다.
- 스토리 구조가 있다.
- 시제가 크게 흔들리지 않는다.
- 같은 단어를 반복해도 자연스럽게 이어진다.
- 감정 표현이 있다.

수정할 답변:

- 한국어로 먼저 생각하느라 멈춘다.
- 과거 경험인데 현재 시제로 말한다.
- 묘사만 길고 사건이 없다.
- 롤플레이에서 대안을 하나만 말한다.
- 14~15번에서 개인 의견 없이 일반론만 말한다.

## 시험 직전 10분

1. 선택 주제 12개 훑기
2. 공원/영화/카페/해변 콤보 1문장씩 말하기
3. 롤플레이 대안 2개 공식 말하기
4. 필러 5개 소리 내기
5. `I have been into`, `I would say`, `which makes me feel` 문장 3개 말하기
