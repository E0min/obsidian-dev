### xAB (A/B 테스트 기반 SNS)

서비스 개요: 두 가지 선택지에 대한 실시간 투표 및 댓글 토론이 가능한 소셜 네트워크 서비스

팀원: 4명 (FE 4명)

기술 스택: TypeScript, Next.js, React Query, Zustand, Supabase, Tailwind CSS

- 팔로워·팔로잉 목록 조회 시 맥락이 끊기는 문제를 해결하기 위해 App Router의 병렬·인터셉팅 라우트(`@modal`)를 결합해 모달로 띄우고 직접 진입 시에는 전체 페이지로 렌더하도록 분기했습니다.

- 팔로우 기능의 데이터 정합을 위해 `follows` 테이블 기반 팔로우 추가·해제 API(POST·DELETE)와 팔로워·팔로잉 목록 조회 라우트를 구현하고, 자기 자신 팔로우 요청을 서버에서 차단하는 검증을 추가했습니다.

- 프로필 도메인을 담당해 프로필 헤더·홈 프로필 섹션 컴포넌트와 사용자별 프로필·게시글 조회 API 라우트를 구현하고, 프로필 페이지 진입 시 사용자 정보와 작성 게시글을 함께 페칭하도록 데이터 흐름을 정리했습니다.

- 메인 피드의 초기 로딩 지연 문제를 해결하기 위해 TanStack Query의 `useInfiniteQuery`와 Intersection Observer 기반 트리거를 결합하여 페이지 단위 분할 호출과 지연 로딩이 적용된 무한 스크롤 피드를 구현했습니다.
