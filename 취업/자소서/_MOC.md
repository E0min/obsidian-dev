# 자소서 — 자기소개서 작업 공간

> 자기소개서는 회사마다 문항이 다르므로 공통 본문을 두지 않는다. 회사별로 `회사별/`에 작성·보관하고, 작성 전 아래 가이드 2종을 먼저 읽는다.

## 작성 전 필수 참조 (2종)

- [[지원동기-작성-가이드]] — 주어 교체 테스트·일상 에피소드 재해석·채용자 고민 중심으로 지원동기 쓰는 법
- [[AI-slop-방지-글쓰기-가이드]] — AI 말투(명언식 한 줄·연속 단문·은유적 정의·대조 부정·금지 어휘) 제거, 사실·수치·기간 중심 문체

## 구조

```
자소서/
├── _MOC.md                    # 이 문서
├── 지원동기-작성-가이드.md      # 영상 요약 캡처 (참조용)
└── 회사별/
    └── {회사명}.md             # 회사별 자소서 (문항별 답변)
```

## 회사별 자소서 작성 흐름

1. `회사별/{회사명}.md` 새로 만들기 (문항을 그대로 헤더로 붙여넣기)
2. [[지원동기-작성-가이드]] + [[AI-slop-방지-글쓰기-가이드]] 먼저 읽기
3. 이력서·포폴의 실제 사실(수치·기술명·기간)을 근거로 문항별 답변 작성
4. 주어 교체 테스트 + AI-slop 금지 패턴 0건 점검 후 마무리
