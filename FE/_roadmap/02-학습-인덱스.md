# 학습 인덱스

> TL;DR: 영역별로 학습할 노트(기존 + 신규), 외부 정전 자료, 맨손 과제, 면접 질문을 한 곳에 모았다. 노트를 끝내면 체크하고 [[01-습관과-원칙]]대로 confidence를 매긴다.

각 노트는 본문을 덮고 `직접 확인 → 스스로 답하기 → 내 프로젝트 연결` 순으로 푼 뒤 본문으로 확인한다. 체크박스는 "한 번 읽음"이 아니라 "빈 화면에서 설명 가능"일 때 체크한다.
